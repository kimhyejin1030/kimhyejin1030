#### 클린 코드와 원활한 협업으로 안정적인 서비스를 만드는 개발자
- Spring Boot, JPA, MyBatis를 활용해 안정적인 REST API를 설계하고 효율적인 데이터베이스 연동 시스템을 구축합니다.
- 팀 프로젝트에서 원활한 소통과 요구사항 조율을 통해 동료들과 함께 복잡한 문제를 해결하며 완성도를 높입니다.
- 단순한 기능 구현을 넘어, 문제의 원인을 끝까지 분석하고 근본적으로 해결하는 과정을 중요하게 생각합니다.
- 문제의 원인을 끝까지 분석하고, 배운 점을 기술 블로그에 꼼꼼히 문서화합니다.
